# HTML-Editor
名称:HTML编辑器
作者：王铭瑄
版本:3.2.2

操作提示:
  1.输入操作
  复制          Ctrl+C
  粘贴          Ctrl+V
  剪切          Ctrl+X
  撤销          Ctrl+Z
  恢复          Ctrl+Y
  查找          Ctrl+f
  2.文件操作
  保存          Ctrl+S
  另存为        Alt+C
  打开          Ctrl+O
  3.代码操作
  预览          Ctrl+R
  检查          Ctrl+C
  格式化        Ctrl+G
  自动修复      Ctrl+F
