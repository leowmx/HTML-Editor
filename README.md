# HTML-Editor

## 基本信息
名称: HTML编辑器<br>
作者: @ 王铭瑄<br>
版本: 3. 6. 0<br>

## 操作说明
1. 输入操作<br>
    复制          Ctrl+C<br>
    粘贴          Ctrl+V<br>
    剪切          Ctrl+X<br>
    撤销          Ctrl+Z<br>
    恢复          Ctrl+Y<br>
    查找          Ctrl+f<br>
2. 文件操作<br>
    保存          Ctrl+S<br>
    另存为        Alt+C<br>
    打开          Ctrl+O<br>
3. 代码操作<br>
    预览          Ctrl+R<br>
    检查          Ctrl+C<br>
    格式化        Ctrl+G<br>
    自动修复      Ctrl+F<br>

4. 其它操作<br>
    见下拉菜单
